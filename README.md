## 프로젝트 명 - SUB-WHERE
순식간에 지나가는 주말에 멀리 여행 가는 것은 부담스럽고, 가까운 곳 여행은 동네 또는 늘 가던 곳만 가게 되는 아쉬움을 덜어보고자 '<strong>수도권 내 지하철로 쉽고 간편하게 이동 가능한 여행지를 추천해주는 사이트' 입니다.

## SUB-WHERE의 주요 기능
1. 여행지 정보 추천
2. 여행 코스 추천
3. 여행 후기 작성
